FAQ

Clear answers to the most common questions.

Is this a full email inbox or just forwarding?

These are full inboxes hosted on Google Workspace (Business Starter class), managed by Rockholt Communications for setup and support.

How do you keep the service “family-only”?

Rockholt Communications is designed for families sharing the Rockholt surname. Access can be managed by invite and onboarding verification (process can be refined over time).

What security features are included?

Google Workspace provides built-in spam and malware protection. We can also encourage strong passwords and multi-factor authentication (MFA) for better protection.

Can we create shared addresses like family@rockholt.com?

Yes—aliases can be configured for shared roles and family functions (events, travel, family, etc.).

What happens if we want to cancel?

Accounts can be offboarded cleanly. If you want continuity, we can plan a transition path so you don’t lose important messages.
